How to Add Yourself to uses.tech

  • Create your website and pages that show when tech and software you use as a web developer
  • On bottom of uses.tech website, click GitHub, "Add Yourself"
  • Create a fork from the GitHub repository
  • Make a change to the src/data/sites.json file and save it
  • Commit your changes and open a pull request
  • Wait for an approval (Wes Box, the owner) or a community member can review and make the merge for approval!
  • Now its Live!!!
